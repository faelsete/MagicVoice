# MagicVoice 🎤

**Высококачественное преобразование текста в речь, просто и быстро.**

Преобразуйте текст в аудио MP3 с использованием нейронных голосов. Без регистрации, без ограничений, без сложностей.

## ✨ Функции

- **Быстрое преобразование:** Для коротких текстов до 9 минут. Вставьте текст, выберите голос, скачайте MP3.
- **Режим аудиокниги:** Для длинных текстов. Система автоматически делит текст на блоки и создает один MP3-файл с использованием параллельной обработки (`pydub`).
- **Несколько движков:** Поддерживает `edge-tts`, `gTTS` и Azure Cognitive Services (опционально).

## 🚀 Установка

```bash
git clone https://github.com/faelsete/MagicVoice.git
cd MagicVoice
pip install -r requirements.txt
python app.py
```
